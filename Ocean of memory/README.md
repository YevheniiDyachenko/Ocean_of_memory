# ОКЕАН ПАМ'ЯТІ

Візуальна новела про ізоляцію, пам'ять і бажання. Створена для **Ukrainian Micro Visual Novel Jam #2**.

## Про гру

Ви — **Олеся**, єдина людиня на орбітальній станції «Ікар-2». За вікном — океан: не вода, а жива субстанція без свідомості в нашому розумінні. База не відповідає вже 86 днів. Коли зовнішня обшивка реєструє дотик, з модуля «Прометей» приходить сигнал — і з ним **Кай**: голос того, кого ви втратили.

Гра досліджує межі між реальністю й ілюзією. Ваші вибори змінюють внутрішній нахил героїні (деформація) і ведуть до різних кінцівок. Є мінігри, посимвольний друк, ефекти глітчу та екран кінцівки зі статистикою й психологічним профілем.

## Як запустити

1. Клонуйте репозиторій або завантажте папку проєкту.
2. Відкрийте `index.html` у браузері (подвійний клік) або запустіть локальний сервер у папці `web`, наприклад:
   - `npx serve`
   - `python -m http.server 8080`
3. Натисніть «Увімкнути термінал», пройдіть титульну сцену, натисніть «Увійти» і грайте.

Збірка не потрібна — проєкт на чистому HTML, CSS і JavaScript.

## Структура проєкту

| Файл / папка | Призначення |
|--------------|-------------|
| `index.html` | Розмітка: термінал, чат, думки, вибори, титул, кінцівка, мінігри |
| `styles.css` | Стилі: ретро-термінал, повідомлення, скролбари, екрани кінцівки |
| `game.js` | Рушій гри: навігація по кроках, друк, звуки, мінігри, збереження стану |
| `story.js` | Контент: кроки історії (`STORY_STEPS`), функція `getStep(id)` |
| `assets/images/` | Фони, аватарки (Олеся, Кай, СИСТЕМА) |
| `assets/audio/` | Фоновий гул, звуки друку, тривоги тощо |
| `assets/fonts/` | Шрифт терміналу (WOFF2 / TTF) |

Історія описується кроками з типами: `line`, `choice`, `branch`, `ending`, `pause`, `horror_shake`, `minigame`. Кроки посилаються один на одного через `next`, `choices[].next`, `nextByDistortion` тощо.
